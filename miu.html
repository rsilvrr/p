Marble It Up! Ultra

3D Physics Platformer for PC, Xbox, PlayStation, Nintendo Switch, and iOS

Level Design – Single Player

Created levels for use in the time-trial single player mode.

These levels are generally linear and needed to be designed with casual players and speedrunners in mind.

Level Design – Multiplayer

Created levels for use in the player-versus-player multiplayer mode.

These levels are generally wide open spaces and needed to be designed with several control schemes in mind (keyboard/mouse, controller, touch).

Gameplay Design

Lead the ideation, iteration, and implementation of the Sumo multiplayer game mode.

UI Design

Designed and implemented the multiplayer scoreboard using Unity's (terrible) UI system.

This scoreboard was designed with multiple platforms in mind and can be collapsed & expanded by the player.

Tools Development

Created a Unity plugin that allows users to create and export custom levels using assets from the game.

Custom levels can be shared and played using the Steam Workshop.
